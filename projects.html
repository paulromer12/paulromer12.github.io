Method Overloading Practice

This Java program demonstrates the concept of method overloading by calculating and printing the average of arrays of different numeric types (short, int, long, and double). It includes: 1. Four overloaded average() methods, each designed to compute the average of arrays of different types. 2. A test program that creates arrays of varying sizes and types and invokes the respective overloaded method to compute the average. 3. A series of printArray() methods to display the elements of each array in a clean, readable format. The program outputs the original elements of the arrays along with their computed averages, ensuring correctness across different data types and array sizes.
